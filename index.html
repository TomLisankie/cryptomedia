What is Cryptomedia?

Cryptomedia, is a nonlinear medium of ownership for hypermedia.

Cryptomedia allows for unique and provable ownership of information, including graphics, audio, video, plain text and any arbitrary file or data.

Cryptomedia is created using the Zora protocol.

A single instance of cryptomedia is a token that has two inseparable components:

  1. Media
  2. Market

The media component features the link to the media, the link to metadata for it, an immutable hash for the media and the metadata. The media linked should be universally free and accessible to the internet. The immutable hashes serve as a tool for the public to verify both links are pointing to the correct files at any point in time.

The market component features the current owner and the list of all bids on that token. The owner has sole access and control over the bids, and can choose to accept any at any time, at which point ownership will transfer to the winning bidder. The current owner may also update the links for the token.

A piece of Cryptomedia itself has its creator, the entity that minted the cryptomedia and receives a defined portion of every future accepted bid. The creator has the right to burn that token if they own it.

Each piece of Cryptomedia also has a unique and immutable token ID, designated at the time of minting in Zora. Each piece is designated its ID sequentially.

Although not part of the Zora protocol, each piece of Cryptomedia has the block number it was created within. This is an important factor which represents the point in time it was created.

Provenance of Cryptomedia can be understood by inspecting its creator, tokenID, blocknumber, the media itself and the history of bids and owners.

Additionally, Cryptomedia has following characteristics:

  • Token-based: An Ethereum-based Non-Fungible Token (NFT)
  • Permissionless: universally usable and censorship resistant
  • Universally accessible: open and available to the internet
  • Valuable: accrue value which is accessible to the owner and creator
  • Internet-native: created within and exists only on the internet
  • Platformless: belonging to no single platform other than itself
  • Composable: can be integrated into other contexts and systems

Cryptomedia is a medium for anyone on the internet to create universally accessible and individually ownable media.

Cryptomedia = [Information + Creator + Owner + Market] = Value
